# Week-four-class-assignment
Class assignment that combines our knowledge of web development of front and back end.
You will have three hours to brainstorm, plan, and build a web application based on one of the provided prompts. The goal is to ideate a solution, implement it using HTML, CSS, and JavaScript, and if applicable, integrate an API to enhance your project. Focus on creating a functional prototype that demonstrates your understanding of the key concepts we've covered throughout the course. At the end of the session, each team will present their project, highlighting the problem they chose to solve, their development process, and how their solution addresses the prompt. Remember to collaborate effectively, manage your time wisely, and most importantly, have fun with the process!



Suggested Project Timeline
20 minutes: Brainstorm with your group
10 minutes: Create your Wireframe Design
90 minutes: Code


Project Requirements:
Must Include:
Wireframe Design: Create a visual blueprint of your web application's layout and structure. Plan the placement and organization of content to ensure a clear and intuitive user experience.
Semantic HTML: Use semantic HTML tags (like header, nav, section, article> and footer) to create a well-structured and accessible layout. Ensure that your HTML code is clean, organized, and easy to understand.
Styling with CSS: Apply custom CSS to style your web application. This includes setting consistent colors, fonts, and spacing. Use CSS properties like flexbox or grid to create visually appealing layouts. Ensure that your design is cohesive and enhances the user experience.


Choose One:
Interactive Elements: Incorporate interactive features usuing javascript that allow users to engage with your application. This could be through buttons, forms, or any other elements that require user input and provide feedback.
API Integration: Use at least one external API to fetch and display dynamic data. Ensure that your API calls are functional and enhance the overall user experience.


Presentation
At the end of the project, you will give a 3-minute presentation to showcase your web application. Walk through the core features of your application, demonstrating how it works and highlighting any key design or technical decisions you made along the way. Be sure to emphasize how your application fulfills the prompt, the technologies you used, and any challenges you overcame during development.



Conclude with a reflection on what you learned during the process and how you might further improve or expand the project if given more time. Keep your presentation clear and concise, focusing on the most impactful aspects of your work.
